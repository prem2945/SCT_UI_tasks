# Coffee Shop Homepage Redesign

This folder contains the complete redesign project for a local coffee shop website homepage.

## Files Included

### 1. `wireframe-low-fidelity.html`
- Low-fidelity wireframe showing the structural layout
- Box-and-line design demonstrating content placement
- Includes design notes explaining layout decisions

### 2. `mockup-high-fidelity.html`
- High-fidelity desktop mockup with full styling
- Modern design with warm coffee-themed color palette
- Interactive elements with hover effects
- Features:
  - Hero section with coffee image
  - Features section
  - Menu preview with product images and rupee pricing
  - About section with mini menu
  - Footer with location information

### 3. `USABILITY_IMPROVEMENTS.md`
- Comprehensive documentation of all usability improvements
- Before/after analysis
- Design system documentation
- Accessibility considerations
- Expected impact metrics

## How to View

1. Open `wireframe-low-fidelity.html` in a browser to see the structural wireframe
2. Open `mockup-high-fidelity.html` in a browser to see the final high-fidelity design
3. Read `USABILITY_IMPROVEMENTS.md` for detailed documentation

## Design Features

- **Modern Aesthetics**: Warm brown and gold color scheme
- **Visual Hierarchy**: Clear content flow and focal points
- **Interactive Elements**: Hover effects and smooth transitions
- **Mobile Considerations**: Responsive design foundation
- **Accessibility**: WCAG compliant color contrasts

## Technologies Used

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- JavaScript (for interactive features)
- Google Fonts (Playfair Display, Inter)
